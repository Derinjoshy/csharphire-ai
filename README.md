# csharphire-ai

AI-powered recruitment automation workflow built with n8n and Gemini API for C# developer hiring.

## Key Features
- AI-powered C# resume screening
- Automated PDF resume extraction
- Gmail approve/deny workflow
- Automatic interview scheduling
- Google Calendar integration
- Automated rejection emails
- Multi-resume processing automation

## Tech Stack
- n8n
- Gemini API
- Gmail API
- Google Calendar API
- Google Drive API
